💡 QUICK TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"Page won't load"
  → Check both servers running: Get-Process node
  → Check ports: netstat -ano | findstr :5000
  → Restart backend: cd backend && npm start
  → Restart frontend: cd frontend && npm run dev

"Login doesn't work"
  → Check credentials are EXACT (case-sensitive for email)
  → Check backend health: http://localhost:5000/health
  → Check browser Network tab for API errors
  → Try clearing browser cache (Ctrl+Shift+Delete)

"Seeing 404 errors"
  → Verify backend is running on port 5000
  → Check Console tab in browser DevTools
  → Look at Network tab for failed requests
  → Ensure .env files are configured

"PWA not showing install prompt"
  → This is normal in development
  → Install prompt shows on production (Vercel)
  → You can still test offline in browser DevTools
