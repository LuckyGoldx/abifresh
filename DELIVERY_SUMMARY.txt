## 🚀 Quick Start (3 Steps)

### Step 1: Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with Supabase credentials
npm run dev
```

### Step 2: Frontend Setup
```bash
cd frontend
npm install
cp .env.local.example .env.local
# Edit .env.local
npm run dev
```

### Step 3: Login
```
URL: http://localhost:3000/login
Email: admin@abifresh.com
Password: SecurePassword123!
```
