## 📞 COMMON ISSUES & SOLUTIONS

| Issue | Cause | Solution |
|-------|-------|----------|
| CORS Error | Wrong CORS_ORIGIN | Update env var to match frontend URL |
| 401 Unauthorized | JWT verification failed | Verify JWT_SECRET matches on backend |
| Database connection fails | Wrong credentials | Check Supabase URL and keys |
| File upload fails | Bucket permissions | Check Supabase storage bucket policies |
| Slow response times | Cold start | Upgrade Koyeb plan or optimize code |
| 502 Bad Gateway | Backend down | Check Koyeb logs, restart service |
| Blank page on mobile | CSS not loaded | Check if PWA cache needs clearing |
